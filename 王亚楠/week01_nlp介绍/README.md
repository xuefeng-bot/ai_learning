机器学习
有监督学习：
  - 核心目标：建立一个模型（函数），来描述输入（X）和输出（Y）之间的映射关系
  - 价值：对于新的输入，通过模型给出预测输出
  - 要点：
    - 需要有一定的数据量
    - 输入输出之间有关联关系
    - 输入输出可以数值化
    - 任务需要有预测价值
  - 应用：
    - 文本分类任务（文本——类别）
    - 机器翻译任务（A语种——B语种）
    - 图像识别任务（图像——类别）
    - 语音识别任务（音频——文本）
无监督学习：给机器的数据没有标注信息，通过算法对数据进行一定的自动分析处理，得到一些结论
  - 常见任务：聚类、降维、找特征值
  - K-Means算法（聚类）
    - 随机初始化：随机选K个点作为中心
    - 分配归属：每个数据点归属于距它最近的中心
    - 更新中心：重新计算每组所有点的平均位置
    - 重复迭代：重复步骤 2-3，直到中心不再变化
  - PCA（降维）（主成分分析）
    - 为什么需要降维（维度灾难、冗余特征、难以可视化）
    - 找到数据中最重要的方向（主成分），把高维数据压缩到低维，同时保留大部分信息
强化学习：通过奖励与惩罚不断试错学习
  - 常见任务：游戏AI、机器人控制、自动驾驶
常见概念：
  - 训练集（train）:用于模型训练
  - 验证集（val）:用于验证对比不同算法的差异
  - 测试集（test）:用于最终测试模型效果，通常与验证集不做区分
  - K折交叉验证：采样成K个子样本，其中k-1个用作训练，一个用作验证。交叉训练验证K次，取平均k次的结果
  - 过拟合：模型失去了泛化能力，模型在训练集上表现好，在测试集上表现差
    - 常见原因：模型复杂度过高、训练数据量不足、特征冗余或噪声多、训练迭代过多、正则化不足
    - 解决方案：简化模型或降低维度、扩大训练数据集（数据增强）、降维或特征选择、早停、加强正则化、Dropout
  - 欠拟合： 模型没能建立起合理的输入输出之间的映射，在训练集中结果就很差
    - 常见原因：模型复杂度不够、特征信息不够、训练不充分、正则化过度
    - 解决方案：增强模型复杂度（更深网络、多项式特征）、加强特征工程、增加训练轮次或用更快的优化器、降低正则化强度
  - 评价指标：准确率、召回率、F1值、TopK、BLEU
  - 偏差（bias）：预测值与真实值的系统性偏差
  - 方差（variance）：对不同数据的敏感程度
    - 低偏差+低方差 ：理想状态 训练模型所追求的
    - 低偏差+高方差：过拟合
    - 高偏差+低方差：欠拟合
  - 超参数（Hyperparameter）：学习率（Learning Rate）、隐藏层数量、每层神经元数量、训练轮数（Epochs）、批次大小（Batch Size）
模型评估：
  - 混淆矩阵：TP（真正例）、TN（真负例）、FN（假负例） 、FP（假正例） 
    - $$Accuracy = \frac{TP+TN}{TP+TN+FP+FN}$$  准确率
    - $$Precision = \frac{TP}{TP+FP}$$  精确率 ： 预测为正的里面，有多少真的是正
    - $$Recall = \frac{TP}{TP+FN}$$  召回率 ： 实际为正的里面，有多少被找出来了
  - F1分数：综合指标
    - $$Recall = 2*\frac{Precision*Recall}{Precision+Recall}$$
    - 比算术平均更重视较小值，两个指标都高才能得高分
    - F1为1代表精确率和召回率都达到了100%
    - 适合正负样本比例悬殊的情况，优于单纯使用准确率
