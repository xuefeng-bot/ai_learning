学习率调度
- 阶梯衰减（step Decay）：每隔固定的epoch将学习率乘以衰减因子

损失函数
1. 均方误差（ MSE ：Mean Squared Error ）损失函数，也称为平方损失函数，是用于回归任务的一种常见损失函数。
该损失函数 用于衡量 模型对于每个样本预测值与实际值之间的平方差异的平均程度。

均方误差损失函数 的数学表达式为：
$$MSE = \frac{1}{n} \sum^n_{i=1}(y_i-\hat y_i)^2$$
其中：
  - $$n$$ 是样本数量
  - $$y_i$$ 是第 $$i$$个样本的 真实标签
  - $$\hat{y_i}$$ 是第 $$i$$个样本的 预测值
- 均方误差损失函数 对差异的平方进行了求和，这使得较大的误差在计算中得到了更大的权重，这也导致它对离群值敏感，因为它会放大离群值的影响。
- 均方误差损失函数 在回归问题中广泛使用，尤其在对误差敏感度较高的情况下。

2. 交叉熵损失函数：$$H(p,q) = -\sum p(x)logq(x)$$   p(x)真实样本标签、q(x)预测概率
交叉熵损失多用于 多分类任务，下面我们通过拆解交叉熵的公式来理解其作为损失函数的意义
假设我们在做一个  n分类的问题，模型预测的输出结果是 $$[x_1,  x_2, x_3, ...., x_n]$$
然后，我们选择交叉熵损失函数作为目标函数，通过反向传播调整模型的权重
交叉熵损失函数的公式 ：
$$\begin{aligned}
loss(x, class) 
&= -log(\frac{e^{x_{[class]}}}{\sum_je^{x_{j}}})\\
&= -x_{[class]} + log(\sum_j e^{x_{j}})
\end{aligned}$$
- $$x$$是预测结果，是一个向量 $$x=[x_1, x_2, x_3, ...., x_n]$$，其元素个数 和 类别数一样多
- class 表示这个样本的实际标签，比如，样本实际属于分类 2，那么 $$class=2$$  ， $$x_{[class]}$$ 就是$$x_2$$，就是取预测结果向量中的第二个元素，即，取其真实分类对应的那个类别的预测值

---
接下来，我们来拆解公式，理解公式：
1. 首先，交叉熵损失函数中包含了一个最基础的部分：$$softmax(x_i) = \frac{e^{x_i}}{\sum_{j=0}^{n} e^{x_{j}}}$$
  softmax 将分类的结果做了归一化：
  -  $$e^{x}$$ 的作用是将 $$x$$ 转换为非负数
  - 通过 softmax 公式 $$\frac{e^{x_i}}{\sum_{j=0}^ne^{x_j}}$$ 计算出该样本被分到 类别 $$i$$的概率，这里所有分类概率相加的总和等于1
2. 我们想要使预测结果中，真实分类的那个概率接近 100%。 我们取出真实类别的那个概率，(下标为 class)：
  $$\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}$$，我们希望它的值是 100%
3.  作为损失函数，后面需要参与求导。乘/除法 表达式求导比较麻烦，所以最好想办法转化成加/减法表达式。最自然的想法是取对数，把乘/除法转化为加/减法表达式 ：
$$log{\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}} = log{e^{x_{[class]}}} - log{\sum_{j=0}^{n} e^{x_{j}}}$$
- 由于对数单调增，那么，求 $$\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}$$ 的最大值的问题，可以转化为求 $$log{\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}}$$ 的最大值的问题。
-  $$\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}$$ 的取值范围是 (0, 1)，最大值为1。 取对数之后，$$log{\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}}$$ 的取值范围为 [−∞,0]，最大值为0
作为损失函数的意义是：当预测结果越接近真实值，损失函数的值越接近于0
所以，我们把 $$log{\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}}$$ 取反之后，$$-log{\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}}$$  最小值为0
这样就能保证当  $$\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}}$$ 越接近于 100%， $$loss=-log(\frac{e^{x_{[class]}}}{\sum_{j=0}^{n} e^{x_{j}}})$$  越接近0。
优化器
1. Adam
累计梯度（越远（时间步，更新次数）的梯度乘了一个越小的系数）：$$m_t \leftarrow \beta_1 m_{t-1} + (1 - \beta_1)g_t  $$
累计平方梯度：$$v_t \leftarrow \beta_2 v_{t-1} + (1 - \beta_2)g^2_t$$
偏差校正：$$\hat{m_t} = \frac{m_t}{\color{blue}1 - \beta^t_1}$$  、 $$\hat{v_t} = \frac{v_t}{\color{blue}1 - \beta^t_2}$$（t次方）（分母随时间步不断变大）
权重更新：$$w_t \leftarrow w_{t-1} - \frac{\color{blue}\hat{m_t}}{\color{blue}\sqrt {\hat{v_t} + \delta}}*\alpha$$
其中：
- $$g$$为梯度，$$\beta_1$$为动量参数（0～1，0.9），$$m$$为累积梯度(初始为0) ，
- $$\beta_2$$为衰减系数（0～1，0.999），$$v$$ 为累积平方梯度(初始为0) ，
- $$\alpha$$为学习率，$$\delta$$为小参数避免分母为0，$$t$$为时间步(初始为1)
优点：
1. 实现简单，计算高效（无指数对数计算），对内存需求少
2. 超参数具有很好的解释性（$$\beta_1$$$$\beta_2$$ 大注重历史，小注重当前），且通常无需调整或仅需很少的微调
3. 更新的步长能够被限制在大致的范围内（初始学习率）
4. 能够表现出自动调整学习率
5. 很适合应用于大规模的数据及参数的场景
6. 适用于不稳定目标函数
7. 适用于梯度稀疏或梯度存在很大噪声的问题

一、 Adam 核心组件的功能分析
Adam 相比于 SGD，主要通过两个指标（$$m_t$$ 和 $$v_t$$）对梯度进行了“加工”，实现了“化腐朽为神奇”的效果：
1. $$m_t$$：动量累加（解决“方向”问题）
- 数学本质：历史梯度的加权平均（动量）。
- 作用：平滑震荡，稳定方向。它不只看当前的梯度，而是参考了历史轨迹。
- 优势：即使某一轮的数据（脏数据或噪声）指向错误的方向，$$m_t$$也能靠历史惯性抵消掉这种随机扰动，防止模型“反复横跳”或频繁调头。
2. $$v_t$$：二阶矩缩放（解决“步长”问题）
- 数学本质：梯度平方的累加和（未中心化的方差）。
- 作用：自适应调节步幅（Learning Rate Adaptive）。
- 优势：
  - 大梯度约束：当梯度很大时，除以 $$\sqrt{v_t}$$会缩小步长，防止更新过猛导致崩盘(梯度爆炸)。
  - 小梯度放大：当梯度很小时，除以一个极小的数能起到放大作用，防止梯度消失。
  - 自适应性：即使你设定的学习率是固定的，Adam 也会根据梯度的大小自动调整实际更新的幅度。
3. 偏置校正（解决“节奏”问题）
- 数学本质：除以$$1 - \beta^t$$进行修正。
- 作用：实现“先快后慢”的训练节奏。
- 优势：随着训练步数 t 的增加，分母变大，整体步幅自然下降。这符合人类寻找目标的逻辑：前期大步流星找方向，后期小步挪动找终点。

二、 文中生动案例总结
为了解释深奥的数学公式，老师使用了两个非常接地气的生活化例子：
4. 找海底捞的例子（解释$$m_t$$动量）
- 情景：你在商场找海底捞，前三个人都指路说“往左走”，第四个人突然说“往右走”。
- 逻辑：你大概率还是会往左走，因为前三个人形成了“共识”，而第四个人可能是不认路的（对应训练集里的噪声或脏数据）。
- 结论：“大众的引导”比“个人的意见”更可靠。累加梯度就是这种大众引导，它不会让你因为一个样本的错误指引就立刻掉头。
5. 下山找景点的例子（解释偏置校正/节奏控制）
- 情景：你在下山寻找一个特定的景点。
- 逻辑：刚下山时，你知道大概方向，可以跑得快一点；但当你接近景点附近时，你必须放慢脚步，仔细寻找，否则容易走过头。
- 结论：科学的训练需要动态的速率。Adam 的数学设计保证了模型在早期能够快速收敛，在后期能够稳健微调。
三、 总结：Adam 为什么更健壮？
1. 容错性高：利用动量抵御脏数据的干扰。
2. 自适应强：无需精细调参，它能根据梯度大小自动匹配最合适的步长。
3. 收敛高效：自带“先快后慢”的进度管理，训练速度和稳定性达到了极佳的平衡。
这也正是为什么 Adam（以及它的变种 AdamW）能成为目前深度学习领域最主流、最通用的优化器的原因。
2. AdamW
权重更新：$$w_t \leftarrow w_{t-1} - (\frac{\color{blue}\hat{m_t}}{\color{blue}\sqrt {\hat{v_t} + \delta}} + \lambda w_{t-1})*\alpha = (1-\lambda \alpha)w_{t-1} - \frac{\color{blue}\hat{m_t}}{\color{blue}\sqrt {\hat{v_t} + \delta}}*\alpha$$
"Weight decay"（权重衰减）是深度学习中常用的一种正则化技术，作用是防止模型过拟合。
通过在损失函数中添加权重衰减项，优化算法（如随机梯度下降）会尝试在最小化训练误差的同时，使模型参数保持较小的值，从而提高模型的泛化能力，防止过拟合
AdamW与Adam相比就是解耦了权重衰减
对于带权重衰减的损失函数 $$\text { Loss }_{\text {total }}=\text { Loss }_{\text {original }}+\lambda \cdot \frac{1}{2}\|w\|_2^2$$， 损失对参数$$w_i$$ 的偏导数为： $$\frac{\partial l}{w_i} + \lambda w_i$$  （其实就是计算完梯度后，在原来梯度上加了一个$$ \lambda w_i$$）

LSTM门控公式（PyTorch约定顺序：i, f, g, o）:

f_t = sigmoid(W_if * x_t + b_if + W_hf * h_{t-1} + b_hf)   遗忘门
g_t =    tanh(W_ig * x_t + b_ig + W_hg * h_{t-1} + b_hg)   候选记忆

c_t = f_t ⊙ c_{t-1} + i_t ⊙ g_t                            细胞状态

标准化Normalization
公式：$$\mathrm{y} = \frac{\mathrm{x} - \mathrm{E}(\mathrm{x})}{\sqrt{\mathrm{Var}(\mathrm{x}) + \epsilon}} * \gamma + \beta$$
其中，  $$\mathrm{E}(\mathrm{x}) = \frac{1}{n}\sum_{i=1}^n x_i,
\quad
\mathrm{Var}(\mathrm{x}) = \frac{1}{n}\sum_{i=1}^n (x_i - \mathrm{E}(\mathrm{x}))^2,$$$$\gamma$$ 和 $$\beta$$ 分别是可学习的缩放和平移参数（常量初始为 $$\gamma=1$$, $$\beta=0$$）。
Normalization（归一化）是深度学习模型中常见的一类操作，目的是通过对激活值或特征张量进行规范化处理，从而缓解梯度消失、加速模型收敛、增强模型的泛化能力。
改变数据分布，将数据分布从一个大区间拉回到一个小区间（不改变数据间的相对大小）
两个可学习的训练参数可以让数据稍微做一做放缩、偏移
输入发生小的变化，输出不应该发生大的变化
Dropout层
作用：减少过拟合
按照指定概率，随机丢弃一些神经元（将其化为零）。其余元素乘以 1 / (1 – p)进行放大
为什么要进行放大：有助于保持内积后的结果与dropout前还在大致同一个范围
如何理解其作用：
- 强迫一个神经单元，和随机挑选出来的其他神经单元共同工作，消除减弱了神经元节点间的联合适应性，增强了泛化能力
- 可以看做是一种模型平均，由于每次随机忽略的隐层节点都不同，这样就使每次训练的网络都是不一样的，每次训练都可以单做一个“新”的模型
启示：计算方式并不是越复杂就越好
dropout在model.train的时候起作用，在model.eval时不起作用
